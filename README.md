## Project Description

A Laravel-based Todo App System that allows users to:

* Create tasks
* Assign categories
* Set due dates
* Mark tasks as completed
* Delete tasks

## Features

* Task Management
* Categories
* Due Dates
* Completed Tasks Section
* Responsive UI Design

## Technologies Used

* Laravel 12
* PHP
* SQLite
* HTML
* CSS
* Blade Templates

## Installation

1. Clone the repository

git clone https://github.com/MernelVelasquez/todo-app.git

2. Install dependencies

composer install

3. Copy environment file

copy .env.example .env

4. Generate application key

php artisan key:generate

5. Run migrations

php artisan migrate

6. Start server

php artisan serve
